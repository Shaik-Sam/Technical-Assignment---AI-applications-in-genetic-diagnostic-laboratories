 AI for Genetic Diagnostic Labs — Technical Assignment (3 Projects)

This repository contains three separate mini-projects, each aligned to a theme commonly encountered in genetic diagnostic laboratories.

- Project 1: `project-1-sample-tracking/` — AI-Based Sample Tracking System
- Project 2: `project-2-ngs-run-failure-mgi/` — AI to Predict NGS Run Failure (MGI)
- Project 3: `project-3-workflow-optimization/` — AI-Based Laboratory Workflow Optimization

Each folder includes:
- A complete solution write-up (problem, workflow, data, model approach, evaluation, deployment, risks)
- Suggested database schemas / event formats
- Minimal starter scaffolding (`src/`, `notebooks/`, `data/` placeholders) you can expand


- Start here: read the `README.md` inside each project folder.
- Data notes: see `data/README.md` in each project (what to collect, privacy, examples).
